# Upgrade guide

## Upgrade from 2.0 to 2.x
Replace the inas PHAR file:

```shell
curl https://raw.githubusercontent.com/nxu/inas/2.0/bin/inas > /usr/local/bin/inas
```

## Upgrade from 1.x to 2.x
Replace the inas PHAR file:

```shell
curl https://raw.githubusercontent.com/nxu/inas/2.0/bin/inas > /usr/local/bin/inas
```
